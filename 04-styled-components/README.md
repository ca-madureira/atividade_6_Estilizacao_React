# Styled Components

Esta versão utiliza **Styled Components** com Theme Provider para gerenciamento de temas.

## Estrutura

- `src/theme/`: Definição dos temas claro e escuro
- `src/components/styled/`: Componentes estilizados reutilizáveis
- Uso de `ThemeProvider` no nível raiz da aplicação

## Como executar

```bash
npm install
npm start
